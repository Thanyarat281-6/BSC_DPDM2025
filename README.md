## Project: Rainfall Prediction using Machine Learning

### Objective
พัฒนาโมเดล Machine Learning เพื่อพยากรณ์ปริมาณฝน  
โดยใช้ข้อมูลสภาพอากาศย้อนหลัง และดัชนีภูมิอากาศ (Climate Indices)

---

### Models Used
-  Neural Network (ANN) & LSTM
-  Huber Regressor
-  K-Nearest Neighbors (KNN)
-  Isotonic Regression

---

### Methodology

1. Data Collection (Rainfall + Climate Index)
2. Data Preprocessing
   - Missing Value Handling
   - Normalization
   - Feature Selection
3. Model Training
4. Cross Validation (K-Fold)
5. Model Evaluation
---

###  Tools Used
- Python (Scikit-learn)
- Google Colab
- Pandas / NumPy / Matplotlib
